## Objective
Define the contract-first compiler surface for Phase 18.

Purpose: satisfy the contract and mapping half of `CMP-01` by introducing a file-based script input model, a committed governed library/mapping catalog, and a reusable Core mapping pipeline that turns ordered script sections into deterministic template instantiation plans without touching CLI rendering/export work.
Output: committed compiler mapping artifacts under `.planning/script-compiler/`, strongly typed Core compilation records, and deterministic mapping validation tests.
